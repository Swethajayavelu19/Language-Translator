**Project Title:** AI Language Translator

---

 Abstract

The AI Language Translator is a modern web application developed using HTML, CSS, and JavaScript. The application translates text between multiple languages using an online translation API. It provides an intuitive Glassmorphism user interface along with advanced features such as voice input, text-to-speech, translation history, dark mode, copy functionality, and downloadable translated text. The project focuses on providing a responsive and user-friendly translation experience.

---

* Develop an interactive language translator.
* Provide translation between multiple languages.
* Create a responsive user interface.
* Improve user experience through modern design.
* Implement voice recognition and speech synthesis.
* Maintain translation history.

---

 Working Process

1. User enters the text.
2. User selects source and target languages.
3. The Translate button sends the request to the Translation API.
4. The API returns the translated text.
5. The translated output is displayed.
6. The user can copy, listen, download, or save the translation history.
