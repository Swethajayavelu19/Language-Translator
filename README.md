 🌍 AI Language Translator

 📌 Project Description

AI Language Translator is a responsive web application developed using **HTML, CSS, and JavaScript**. It enables users to translate text between multiple languages through an online translation API. The application also provides voice input, text-to-speech, translation history, and a modern Glassmorphism user interface.

---

 🚀 Features

* 🌐 Multi-language Translation
* 🌍 Supports Multiple Languages
* 🔄 Language Swap
* 🎤 Voice Input
* 🔊 Text-to-Speech
* 📋 Copy Translated Text
* ⬇️ Download Translation as TXT
* 🌙 Dark / Light Mode
* 📜 Translation History
* 🔢 Character Counter
* ✨ Glassmorphism User Interface
* 📱 Responsive Design

---

 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* MyMemory Translation API
* Web Speech API

 ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open **index.html** using any modern web browser or VS Code Live Server.
4. Enter text and choose the source and target languages.
5. Click **Translate** to view the translated result.
